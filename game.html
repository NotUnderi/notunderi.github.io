Project Details

This platformer game was developed as part of a product development course at Turku University of Applied Sciences (TUAS). I was responsible for the programming, including game mechanics, physics, and level design.

The game features a 2D side-scrolling environment with collectable coins and obstacles.

Development window was extremely short, so the game was developed to only have the most important features and mechanics, such as player movement, jumping, and coin collection, music handling, pause menus and basic settings.

The project was a collaborative effort, with team members contributing to art, sound design, and testing.

The source code is available on GitHub.

Project Details

Rekkaribotti is a Discord bot designed to search and cache Finnish license plates, providing detailed information about vehicles. The bot simplifies access to vehicle data for users in a Discord server.

The bot integrates with Biltema's API to fetch vehicle information and uses caching to optimize performance, reduce redundant API calls and provide statistics.

The project is open-source and can be found here.

Gallery

Key Features

  • Search Finnish license plates directly in Discord
  • Cache results to improve performance
  • Fun statistics about seen cars
  • Simple and user-friendly commands

Software Stack

  • Python
  • Discord.py
  • SQLite for caching
